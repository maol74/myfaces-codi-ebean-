
This is a sample project for testing out and providing a project template for following stack.

Apache MyFaces 2.x
Apache Codi as CDI enhancement and CDI JSF integration library
Apache OpenWebbeans as CDI Provider

Ebeans as ORM Layer
H2 as embedded database

You can kickstart the project after you have installed maven 3 by following commands
mvn clean install jetty:run-exploded


After the startup point your browser towards
http://localhost:9090/
